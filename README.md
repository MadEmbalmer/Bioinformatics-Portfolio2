## Featured Research Project

### 1. RNA 3D Structure Prediction via Geometric Deep Learning (Generative Modeling)

- **Language:** Python  
- **Methods:** E(n)-Equivariant Graph Neural Networks (EGNN), diffusion-based generative modeling, torsion-angle representation, internal coordinate reconstruction, structural clustering  
- **Focus:** Geometric inductive bias for structured generative modeling under limited biological data  

Repository:  
https://github.com/MadEmbalmer/Bioinformatics-miniportfolio1  

This project models RNA backbone torsion angles using an equivariant graph neural network and trains a diffusion model over torsion space to generate 3D structures.

It demonstrates:

- Equivariance principles and geometric inductive bias  
- Diffusion-based generative modeling  
- Angular loss design and coordinate reconstruction  
- Structure sampling and clustering  
- Modeling under small-data constraints
